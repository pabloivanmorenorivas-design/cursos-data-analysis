# 📊 Laboratorios de Análisis de Datos

Este repositorio está diseñado para centralizar, documentar y hacer un seguimiento estructurado de mi progreso técnico en la manipulación, limpieza y análisis de datos. 

Para optimizar el flujo de trabajo y eliminar la fricción de configuraciones locales, todo el código y los entornos de ejecución se gestionan mediante **Google Colab** y se sincronizan directamente a este espacio en la nube.

---

## 🗂️ Estructura del Repositorio

* **`curso-kodree/`** — Prácticas dinámicas, retos lógicos y libretas individuales creadas desde cero para cada módulo del programa.
* **`curso-arturo-hernandez/`** — Estructura predefinida de notebooks analíticos del curso, completados y ejecutados paso a paso.

---

## 🛠️ Tecnologías y Herramientas en Práctica
* **Lenguaje:** Python 3
* **Librerías Clave:** Pandas, NumPy, Matplotlib, Seaborn
* **Entorno:** Google Colab Cloud Ecosystem
* **Control de Versiones:** Git & GitHub

---

## 🚀 Cómo ejecutar los laboratorios
Cada libreta (`.ipynb`) dentro de este repositorio incluye un botón superior de **"Open in Colab"**. Al hacer clic, el entorno se iniciará automáticamente en la nube listo para producción y pruebas.
